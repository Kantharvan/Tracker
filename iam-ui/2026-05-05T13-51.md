- **Diagnosed broken `/clear` → Obsidian pipeline:** The `prompt` hook type wasn't generating session summaries before context clear, leaving all Obsidian notes with "No compact summary available" placeholders
- **Root cause:** `/clear` executes immediately without allowing the `prompt` hook time to generate and save a summary to `~/.claude/scripts/.last-session-summary`
- **Solution implemented:** Replaced broken `prompt` hook with two sequential `command` hooks in `settings.json`:
  - `generate-session-summary.py` — reads conversation JSONL transcript, calls Claude Haiku API via localhost proxy, writes summary to `.last-session-summary`
  - `obsidian-session-note.sh` — existing script picks up summary and writes to `~/obsidian-vault/Claude Sessions/`
- **Key files created/modified:** `~/.claude/scripts/generate-session-summary.py` (new), `settings.json` (hook config updated)
- **Tested end-to-end:** Both scripts verified working against current session; next `/clear` command will now properly generate and persist session summaries to Obsidian notes
